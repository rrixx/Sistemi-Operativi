**INTERFACCIA UTENTE** 
il SO mette a disposizione due interfacce:
- **INTERPRETE COMANDI** (shell) per un interazione da linea di comando 
- **INTERFACCIAN UTENTE** (GUI) per un interazione mediante elementi grafici sul desktop 

**INTERFACCIA PROGRAMMATORE** 
l'interfaccia verso i processi consiste nell'**API** (Application Programming Interface) che rende disponibili le [[system call]] che possono essere di diverse classi :
- di gestione processi
- di gestione di file e dispositivi 
- di gestione di informazioni di sistema
- di comunicazione/sincronizzazione tra processi 

**PROGRAMMA DI SISTEMA**= programma che chiama le system call
