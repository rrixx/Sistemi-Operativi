la memoria secondaria serve per: 
- rappresentare lo spazio logico in blocchi
- allocare e de-allocare questi blocchi
- gestire lo spazio libero 
- scheduling delle operazioni sul disco 

solitamente la gestione dei file segue quella della memoria secondaria e la memoria secondaria è indipendente dalla gestione dei file 

per quanto riguarda il File System il SO:
- realizza il concetto di file come unità memorizzabile
- crea le strutture come cartelle e directory da manipolare 
- associa i file con i dispositivi di memorizzazione secondaria 

concetto di **FILECENTRICITA**, ovvero qualsiasi risorsa del sistema può essere rappresentata da un oggetto **file**, ciò rappresenta un livello di astrazione universale
