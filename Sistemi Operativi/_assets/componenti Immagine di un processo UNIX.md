sono:
- **process structure**: elemento della Process Table associato al processo  
- **user structure**: dati necessari al kernel per gestire il processo 
- **text**: elemento della Text Table associato al processo kernel residente
- **area di codice**: codice del programma in esecuzione
- **area dati globali utente**: variabili globali del programma eseguito dal processo
- **stack, heap di utente**: aree dinamiche associate al prog eseguito
- **stack del kernel**: stack di sistema associato al processo per le chiamate a system call
