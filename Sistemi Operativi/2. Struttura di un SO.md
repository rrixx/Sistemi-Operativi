le componenti fondamentali d un SO sono: 
1. [[gestione dei processi]]
2. [[gestione della memoria centrale]] 
3. [[gestione della memoria secondaria e file system]]  
4. [[gestione dispositivi di Input-Output]]
5. [[protezione e sicurezza]] 
6. [[interfaccia utente-programmatore]] 

esistono varie strutture per un Sistema Operativo 
1. **monolitica** -> 1 solo modulo eseguibile a basso costo ma poco scalabile 
2. **modulare** -> componenti del SO organizzati in moduli con interfacce ben definite, tuttavia la problematica maggiore è quella della velocità di attraversamento dei vari moduli 
3. **microkernel** -> introduzione di un  [[kernel]] (nucleo)  con struttura ridotta a poche funzionalità di base, caratterizzato da affidabilità, possibilità di estensione e personalizzazione, tuttavia essendoci molte system calls non risulta molto efficiente   
   esistono anche i [[kernel ibridi]] e i [[kernel modulari]] 
4. **macchine virtuali** -> virtualizzano sia hardware che il kernel del SO, tramite la [[virtualizzazione]]
