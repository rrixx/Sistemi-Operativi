componente del SO che fornisce i meccanismi di aceso e memorizzazione 

concetti astratti:
- [[file]] :unita di memorizzazione
- [[direttorio]] : insieme di file
- **partizione**: insieme di file associato ad un dispositivo fisico 

il File System ha una struttura a componenti organizzati su vari livelli

![[Screenshot 2026-03-17 095357 2.png|283]]

- **struttura logica**: presenta una visione astratta delle info di file, directrory e partizioni
- [[accesso]] : definisce i meccanismi per accedere al contenuto 
- [[organizzazione fisica]]: rappresenta file e directory sul dispositivo (allocazione e struttura logica)
- **dispositivo virtuale**: rappresenta una vista astratta al dispositivo (sequenza di blocchi)

_file system mounting_: processo secondo il quale il SO rende accessibili i file e le directory 

_**FILE SYSTEM E PROTEZIONE**_
il proprietario/creatore controlla da chi e quali azion sono sonsentite sui file 
le **modalità di accesso** sono: **read, write, execute**
